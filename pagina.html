Python e HTML Juntos: Uma Dupla Poderosa na Web

Entenda como essas linguagens se complementam no desenvolvimento web.

O que é HTML? (A Estrutura da Página)

HTML (HyperText Markup Language) define a estrutura de uma página. Ele é como o projeto de uma casa.

<!DOCTYPE html>
<html>
<head>
   <title>Minha Primeira Página</title>
</head>
<body>
   <h1>Bem-vindo ao meu site!</h1>
   <p>Este é um parágrafo de exemplo.</p>
   <a href="https://www.google.com">Ir para o Google</a>
</body>
</html>

O que é Python? (A Lógica por Trás da Cena)

Python é uma linguagem de programação usada no backend, onde processa dados, interage com banco de dados e executa a lógica do sistema.

nome = "Mundo"
mensagem = f"Olá, {nome}!"
print(mensagem)

Como Eles Trabalham Juntos?

Com frameworks como Flask ou Django, Python processa as requisições e retorna HTML personalizado ao navegador.

Exemplo Prático com Flask

from flask import Flask, render_template_string
from datetime import datetime

app = Flask(__name__)

@app.route('/')
def home():
    nome_usuario = "Visitante"
    hora = datetime.now().strftime("%H:%M:%S em %d/%m/%Y")
    html_content = f"""
    <!DOCTYPE html>
    <html>
    <head><title>Saudação</title></head>
    <body>
    <h1>Olá, {nome_usuario}!</h1>
    <p>A hora atual é: {hora}</p>
    </body></html>
    """
    return render_template_string(html_content)

app.run(debug=True)

Templates (Jinja2)

Você também pode usar arquivos HTML com variáveis, facilitando a reutilização de código:

<h1>Olá, {{ nome }}!</h1>
<p>Agora é {{ data_hora }}.</p>
